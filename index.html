What is SMSLab?

SMSLab is an independent academic project designed to help researchers, PhD students, instructors, and students create interactive social media stimuli for survey experiments. The tools generate customizable HTML stimuli that can be copied into Qualtrics and used to track participant interactions through Embedded Data.

Support SMSLab

SMSLab is an independent project developed by a PhD student with limited research funding. If you find these tools useful, you can support the project with a small donation. Contributions help cover domain, hosting, maintenance, development time, and the creation of new generators and guides.
